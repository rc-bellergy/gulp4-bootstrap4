# What is it?

Useing gulp 4 auto build bootstrap 4 starter template.
Included:
- bootsrtap 4.4.1
- fortawesome 5.12.1
- jquery 3.4.1

Created by michael@designquest.com.hk

## Install
    npm install
    
## Start
Start development server on http://localhost:8080<br>
Browser will auto reload when js, scss, html changed.

    gulp

## Build

    gulp build
